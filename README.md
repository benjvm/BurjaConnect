# Acerca de BurjaConnect
Bienvenidos a nuestro proyecto de Diseño de Interfaces Web (DIW), BurjaConnect es un sitio web desarrollado y diseñado por tres estudiantes de segundo año de DAW.  
El proyecto consiste en un Sitio Web diseñado para el uso de jóvenes de Burjassot, lugar en donde pueden ver las últimas noticias sobre la zona, los proximos eventos e incluso un apartado de "Que tanto conoces sobre Burjassot?"       


## Tecnologías usadas
- HTML5  
- CSS3  
- React
- Google Maps API
- FullCallendar API

## Dependencias necesarias para un correcto funcionamiento
NPM:     
- npm install @fullcalendar/react   
- npm install @fullcalendar/daygrid   
- npm install react-player     
- npm install react-router-dom  
- npm i react-router          
- npm install react-router-hash-link      
- npm install @fullcalendar/core
